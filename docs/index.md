# Welcome to Will Group Documentation

This is the central hub for the documentation of the Will Group project, a simple Next.js application representing an Arabic landing page.

## Project Architecture

The project relies on Firebase Firestore for dynamic data fetching, and is built using:
- **Next.js 15**
- **TypeScript**
- **Tailwind CSS 4.0**
- **Firebase Firestore**

## Useful Commands

- Run the main Next.js app locally: `npm run dev`
- Run MkDocs server locally: `mkdocs serve`

For more details, please refer to the `README.md` or `GEMINI.md` files located in the project root.
